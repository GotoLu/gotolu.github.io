1. 改头换面的开始

打开hexo的原始封面迎面扑来的满满都是hexo的气息，虽然画风不错，但这终归不是咱们自己的风格不是。这个时候的我们青春弥漫，积极向上，斗志也很是昂扬的，你们看看下图是不是很容易就能感受这种氛围，虽然只是简简单单几行文字仅此而已。哦？你说感觉不到，那可能是你老了吧，嘿嘿

没错，我们接下来的任务就是按照上图的要求修改内容，让它尽可能的符合我们的风格，满足我们的需求，当然哪方面需求自己斟酌就可以了。上篇中讲到博客的文件夹，现在立马戳进去，看到’_config.yml’了没，赶紧用sublime(没装，好吧，反正sublime也没给广告费，爱用啥用啥)打开他的，开头你就能找到这些个配置项了。

title: 给你的博客起个霸气十足的外号吧，外强中干的名字吧(现在啥都没有，名字再响亮不也只得外强中干)
subtitle: 子标题来了，怕你对现在起的名字感到不够霸气，所以现在你可以在你的子标题中再次一站风(风)采(骚)了，看看服务多周到啊！
description: 介(吹)绍(嘘)自己的时候到了，不管是个人还是博客，多介绍介绍，让别人对你(博客)多些了解，我猜人家估计对你不感兴趣，对你的博客更感兴趣吧。
author: 你的名字，给自己起个响亮的笔名也是不错的选择，或者之前就有的那就更好啦。
language: 毫无疑问的我选了zh-CN，当然这个随意，对于我这种英语渣渣，还是老实消停选中文为妙。
timezone: 这个…这个..，这个B装不了了，赶紧跑。回头：不设置也没关系的，反正我没设置，就是这么任性的说！

顺带让你们看看我设置后的效果，是不是清新脱俗、与众不同呢

哟哟哟！是不是发现了和你们不一样的地方，有没有感觉看得让你心旷神怡，仰慕不已呀(这样真的好吗，哈哈哈)。不要着急，这就是咱们今天内容中重点中的重点，飞机中的战斗机—主题更换。

2. 恰同学少年，风华正茂

帅气(美丽)的你，就该有一个别致的主题。现在给你们推荐一款主题：jacman，毕竟目前我用的就是它了，蓝蓝的背景总能让人心情愉悦。
现在是时候来一波认真的了，还是上面那个配置文件，一直反倒最后，你就能看到下图代码

红色的字多显眼，别说没找着，没找到的话，还记得上篇部署到Github中修改的配置吗，没错就是deploy，theme就在它上面，如果你没改动过的说。这里你只需要填写你要更改主题的名称，比如我们现在要更改的—jacman。

重头戏来了，在Github里clone jacman主题的项目。
git clone https://github.com/GotoLu/jacman.git themes/jacman
将jacman项目clone到themes目录下。
接下来我们要做的是将我们博客的主题更换成jacman，进入jacman目录，cd themes/jacman，更换主题git pull origin master，记得每次都要备份主题配置文件。其实这个时候本地预览已经可以看到更换主题后的效果了，查看本地预览更新执行：
hexo s -g
在这里，首先感谢一下jacman的制作者，感谢他上传了这个漂亮舒适的主题。
不过主题难免具有jacman主题制作者的风格，我们可以适当调整，让它更为适合我们。

3. 修改主题相关，让它丰富多彩

在themes/jacman中打开配置文件’_config.yml’，开始配置我们的主题吧。
导航栏设置：

menu: #导航栏
Home: #主页
Archives: #归档
About: #关于自己
Tools: #网络工具
Share: #资源共享

上面的信息根据自己的需求设置吧！

侧边栏挂件设置：

widgets: 
- github-card  #github名片
- category  #目录
- tag  #博文标签
- links  #友情链接
- douban  #豆瓣
- rss  
- weibo #微博

jacman提供了以上七个侧栏挂件,如下图：

theme_image和theme_color的设置：

#### Image
imglogo:
enable: #肯定是true啊，不然你都没法看都image
src: #logo图片，可以自己更换，在jacman/source/img文件夹中
favicon: #网站图片，同上
apple_icon: #同上
author_img: #自己的图片，更换地址同上
banner_img: #标题图片
### Theme Color
theme_color: #主题颜色
theme: ‘#2ca6cb’ ##the defaut theme color is blue
更改完，你会发现主题完全不同了。

个人信息设置

author:
intro_line1: “Hello ,I’m goto8lu Page in Google.”
intro_line2: “This is my blog,believe it or not.”
weibo: #微博数字ID
weibo_verifier: 020af082 #微博秀侧栏验证码
tsina: #新浪微博ID，作为分享使用
douban: #豆瓣ID
zhihu: #知乎ID
email: #个人邮箱
twitter: #Twitter ID
github: #github ID
facebook: #facebook ID
linkedin:
google_plus: “Google plus ID”，不要忘记双引号
stackoverflow: stackoverflow ID
